# Landing Release Checklist

Checklist obrigatorio antes de publicar em producao.

## Visual (obrigatorio)

- [ ] Logo da chave correta no header.
- [ ] Hero com imagem correta.
- [ ] Grid de projetos com 6 imagens corretas.
- [ ] Rodape com texto de regiao correto.
- [ ] Sem texto quebrado (acentuacao).

## Funcional

- [ ] Links do menu funcionando.
- [ ] Botao de orcamento abre modal.
- [ ] Envio de formulario funcionando.

## Tecnico

- [ ] CI da landing passou (UTF-8 + referencias + assinatura + lock de hash).
- [ ] Verificado em aba anonima apos deploy.
- [ ] Cache validado (sem asset antigo).
