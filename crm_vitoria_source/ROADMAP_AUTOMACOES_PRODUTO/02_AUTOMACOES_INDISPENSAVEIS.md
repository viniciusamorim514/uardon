# Automacoes Indispensaveis

## Novo Orcamento

Quando um orcamento chegar pelo formulario:

- criar lead;
- origem: Instagram;
- etapa: Orcamento recebido;
- status: Novo;
- criar alerta no sino;
- criar tarefa "Responder orcamento";
- sugerir mensagem de WhatsApp.

## Lead com Contato Feito

Quando lead mudar para "Contato feito":

- criar tarefa "Marcar briefing";
- sugerir link de agenda;
- sugerir mensagem de confirmacao.

## Projeto Criado

Quando um projeto for criado:

- criar etapas padrao;
- criar tarefas iniciais;
- criar checklist do projeto;
- sugerir mensagem de boas-vindas ao cliente.

## Etapa de Projeto Avancada

Quando etapa do projeto mudar:

- criar proxima tarefa;
- sugerir mensagem ao cliente;
- atualizar status visivel do projeto;
- registrar data da mudanca.

## Parcela Proxima do Vencimento

Quando parcela estiver perto de vencer:

- criar alerta no sino;
- criar tarefa de cobranca amigavel;
- sugerir mensagem pronta para WhatsApp.

## Parcela Atrasada

Quando parcela atrasar:

- criar alerta forte;
- destacar no painel financeiro;
- marcar cliente com pendencia;
- sugerir mensagem de cobranca amigavel.

## Cliente Sem Contato

Quando cliente ficar muito tempo sem contato:

- criar tarefa de follow-up;
- sugerir mensagem de relacionamento;
- destacar em Clientes 360.
